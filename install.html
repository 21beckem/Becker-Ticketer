Install Becker-Ticketer Extension

Instructions for Google Chrome

  1. Click here to download the extension: BeckerTicketerDownloader
  2. Copy and paste chrome://extensions in the address bar and press Enter.
  3. Enable Developer mode (toggle it on in the top-right corner).
  4. In the file explorer window that opened up after the extension was donloaded, click and drag the BeckerTicketerExtension folder to the exensions page.
  5. Reload Genesys

Instructions for Microsoft Edge

  1. Click here to download the extension: BeckerTicketerDownloader
  2. Copy and paste edge://extensions in the address bar and press Enter.
  3. Enable Developer mode (toggle it on in the lower-left corner).
  4. Click on Load unpacked and select your extension's folder.
  5. In the file explorer window that opened up after the extension was donloaded, click and drag the BeckerTicketerExtension folder to the exensions page.
  6. Reload Genesys
